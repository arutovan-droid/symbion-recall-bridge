# symbion-recall-bridge

Blind bridge between session end and Alchemist ingress.

## Laws
- Never reads raw text
- Never interprets human meaning
- Never makes ontological decisions
- Stores only distilled recall snapshots
- Delivers minimal recall_context to Alchemist

## v0.1 scope
- Hot memory only
- Max 3 snapshots per operator
- Local filesystem only
- No warm/cold tiers yet
